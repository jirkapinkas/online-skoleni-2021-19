
Instalace yarn:

    npm i -g yarn

Vytvoreni react aplikace:

    npx create-react-app eshop-frontend

Spusteni aplikace pro vyvoj:

    yarn start

Stazeni dependencies:

    yarn

Build aplikace pro produkci:

    yarn build
